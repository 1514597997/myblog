Mybatis延迟加载(懒加载)

发表于 2025-11-06 分类于 后端
  • 在Mybatis中使用多表联查时，可能会有不需要的数据被相关查询，这时就可以使用该技术优化节省查询次数。
    • 列如查询订单信息，默认关联用户、部门、岗位相关信息，可设置只有需要获取时才查询用户、部门、岗位相关信息

使用示例:

  • 先mybaits-config.xml全局配置懒加载
    •

    //mybatis-config.xml 开启懒加载











  • 用户Mapper:
    •
